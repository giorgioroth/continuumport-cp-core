# Changelog

All notable changes to CP-Core will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-14

### Added
- Initial release of CP-Core v1.0
- Minimal semantic continuity schema (5 core fields)
- JSON Schema validation for v1.0
- Version immutability guarantee
- Design philosophy: Portability > Fidelity
- Usage principles documentation
- Divergence Test specification
- Version evolution policy

### Core Principles
- The less you specify, the more you constrain
- Portability beats fidelity
- Direction beats determinism
- Boundaries beat specifications
